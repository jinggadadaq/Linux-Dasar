# Linux Server Lab

Repository ini berisi panduan praktik **Linux dasar, Web Server Apache, Virtual Host, dan WordPress**.

## Materi
- Pengaturan dasar server (hostname, timezone, IP statis)
- Perintah dasar Linux
- Manajemen file, user, dan grup
- Instalasi Apache

## Cara Pakai
Ikuti langkah-langkah konfigurasi secara berurutan sesuai dokumentasi modul.

## Alat dan Bahan
- Ubuntu Server
- Akses root/sudo
- Koneksi internet
